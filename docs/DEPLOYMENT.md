# Deploy

1. Instale dependencias com `npm install`.
2. Valide dados com `npm run validate:data`.
3. Rode testes com `npm test` e `npm run e2e`.
4. Gere producao com `npm run build`.
5. Publique em um provedor Next.js compativel.

Supabase e opcional e deve ser configurado futuramente por variaveis de ambiente.
