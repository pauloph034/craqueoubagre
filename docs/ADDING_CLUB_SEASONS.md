# Inclusao de Novos Elencos

1. Adicione o clube em `clubs.json`.
2. Adicione o clube-temporada em `club-seasons.json`.
3. Adicione ao menos 15 jogadores em `players.json`, incluindo goleiro e posicoes variadas.
4. Use `dataConfidence: "demo"` quando um dado nao tiver verificacao historica.
5. Rode `npm run validate:data`.

Nao use escudos oficiais, fotos oficiais ou ratings copiados de jogos comerciais.
